release, bringing powerful new features to this AI-powered English dictation and
learning tool. This update introduces Exam Mode, database integration, custom
content capabilities, and comprehensive admin controls to take your learning
experience to the next level.

## Database Integration

Spello now integrates with Supabase Postgres on Vercel, bringing powerful data
management capabilities:

- Secure cloud storage for user progress and learning materials
- Seamless synchronization across devices
- Improved performance and reliability
- Foundation for upcoming user accounts and progress tracking

## Why Choose Spello v0.2.0?

The latest version of Spello offers an even more comprehensive solution for
improving English spelling, typing, and listening skills. With the addition of
Exam Mode, custom content capabilities, and administrative tools, Spello is now
suitable for a wider range of educational contexts, from self-directed learning
to classroom use.

The database integration ensures your progress is safely stored and accessible
across devices, while the custom content features allow you to focus on exactly
the areas you need to improve.

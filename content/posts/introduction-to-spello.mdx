[Spello](https://spello.redmojo.au) is an AI-powered English dictation and
learning tool designed to improve spelling, typing accuracy, and listening
skills through interactive dictation exercises. In this post, we will walk you
through the key features and functions of Spello.

## Working Modes

Spello offers three distinct working modes to accommodate different learning
styles and needs:

### 1. **Standard Mode**

In Standard Mode, users listen to a dictated sentence and type it entirely from
memory without any help.

<img
  src='/images/posts/spello-v2/spello-standard-mode-v5.png'
  alt='Spello Standard Mode'
  className='rounded-xl border border-gray-300'
/>

### 2. **Help Mode**

In Help Mode, users receive instant visual feedback while typing:

    - ✅ Correct characters are highlighted in green
    - ❌ Incorrect characters are highlighted in red

<img
  src='/images/posts/spello-v2/spello-help-mode-v5.png'
  alt='Spello Help Mode'
  className='rounded-xl border border-gray-300'
/>

### 3. **Cheating Mode**

    Similar to Help Mode, Cheating Mode provides real-time feedback by
    highlighting correct and incorrect characters while typing. However, this
    mode also displays the full sentence.

<img
  src='/images/posts/spello-v2/spello-cheating-mode-v5.png'
  alt='Spello Cheating Mode'
  className='rounded-xl border border-gray-300'
/>

### 4. [**Exam Mode** (v0.2.0 new feature)](./spello-release-v0.2.0)

In this mode, users must complete the entire dictation in a single consolidated
text area for input instead of word-by-word inputs.

<img
  src='/images/posts/spello-v2/spello-exam-mode-v5.png'
  alt='Spello Exam Mode'
  className='rounded-xl border border-gray-300'
/>

### 5. [**Study Mode** (v0.4.1 new feature)](./spello-release-v0.4.1)

Study Mode helps users improve comprehension and retention by breaking down
complex sentences into smaller, meaningful thought units. This mode allows
learners to focus on understanding the structure and meaning of each part before
moving to the next.

For example, instead of presenting the full sentence all at once:

> _The celebrated theory is still the source of great controversy._

Study Mode reveals it step-by-step:

- _The celebrated theory_
- _The celebrated theory is still_
- _The celebrated theory is still the source of great controversy._

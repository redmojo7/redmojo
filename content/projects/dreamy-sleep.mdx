**Dreamy Sleep** is a lightweight, calming web application designed to provide
continuous background audio for rest, relaxation, and sleep. It is especially
useful for nighttime routines—such as winding down after a long day, light
meditation, or playing gentle sounds while a newborn is settling or sleeping.

The app focuses on simplicity, aesthetics, and accessibility, featuring ambient
soundscapes, soft lullabies, a clean glassmorphic interface, smooth animations,
and an optional sleep timer to stop playback once sleep is established. Dreamy
Sleep runs entirely in the browser, requires no login or backend services, and
avoids interruptions, making it a quiet alternative to mainstream media
platforms.

## Key Features

- **Curated Sleep Sounds**  
  Multiple ambient sounds organized by language and theme.

- **Language‑Based Categories**  
  Easily switch between:
  - Universal sounds
  - Cantonese children’s songs
  - Japanese children’s songs

- **Sleep Timer**  
  Set a custom timer with hour and minute precision. Audio fades out
  automatically when the timer completes.

- **Fine‑Grained Volume Control**  
  Powered by the Web Audio API for smooth, real‑time volume adjustments.

- **Calming Visual Design**
  - Glassmorphic UI elements
  - Subtle animated starry‑night background
  - Soft typography and gentle color palette

- **Responsive Layout**  
  Optimized for both desktop and mobile devices.

## Technology Stack

- **HTML5**  
  Semantic markup and native audio elements.
- **CSS3**  
  Custom properties, Flexbox, Grid, animations, and backdrop filters.
- **Vanilla JavaScript (ES6+)**  
  Clean, dependency‑free logic.
- **Web Audio API**  
  `AudioContext` and `GainNode` for precise audio control.
- **Google Fonts**  
  Quicksand font family for a soft, approachable feel.

## Getting Started

Dreamy Sleep runs without any build tools or frameworks.

### Local Development

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Deployment Options

### GitHub Pages

1.  Fork https://github.com/redmojo7/Dreamy-Sleep
2.  Open **Settings → Pages**
3.  Select the `main` branch as the source
4.  Visit:

```text
https://your-username.github.io/Dreamy-Sleep
```

### Vercel

```bash
npm i -g vercel
vercel
```

Or connect the repository directly via https://vercel.com.

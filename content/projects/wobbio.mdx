[Wobbio](https://wobbio.redmojo.au) is a modern full‑stack web application
starter designed to showcase best practices in building scalable, type‑safe
React applications with clean architecture, robust routing, and reliable data
persistence.

Wobbio provides a practical foundation for production‑ready apps, combining a
powerful frontend stack with a PostgreSQL backend, structured routing, and
developer‑friendly tooling. It is well‑suited for rapid prototyping, internal
tools, and real‑world CRUD‑style applications.

## Key Features

- **Modern React Architecture**: Built with React and TanStack Router for
  predictable, type‑safe, file‑based routing.
- **Full‑Stack Ready**: Includes server functions and database access, making it
  easy to build real applications end‑to‑end.
- **PostgreSQL Integration**: Uses PostgreSQL for persistent storage, with
  automatic schema initialization.
- **Demo Application Included**: Ships with a Milk Tracker demo to illustrate
  data modeling, server routes, and UI patterns.
- **Clean State Management**: Demonstrates both route loaders and TanStack Store
  for local and derived state.
- **Scalable Data Fetching**: Supports TanStack Query for efficient, cache‑aware
  data loading.

## Technologies

- **React**: Component‑based UI with modern hooks.
- **TanStack Router**: Type‑safe, file‑based routing system.
- **TanStack Query**: Powerful server‑state management.
- **TanStack Store**: Lightweight and predictable client state management.
- **PostgreSQL**: Reliable relational database for production workloads.
- **Tailwind CSS & Shadcn UI**: Utility‑first styling with accessible UI
  components.
- **Vitest**: Fast, modern testing framework.
- **ESLint & Prettier**: Consistent code quality and formatting.

## Getting Started

To run Wobbio locally:

```bash
pnpm install
pnpm start
```

To build for production:

```bash
pnpm build
```

## Testing

Wobbio uses **Vitest** for unit and integration testing:

```bash
pnpm test
```

## Summary

Wobbio is designed as a practical, opinionated starting point for modern web
applications, bringing together the TanStack ecosystem with real‑world backend
integration. It focuses on clarity, maintainability, and patterns that scale
from small demos to production systems.
